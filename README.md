# 🗺️ Leaflet Google Sheets Map

Διαδραστική εφαρμογή χαρτογράφησης με χρήση **Leaflet.js** και **Google Sheets** ως πηγή δεδομένων.  
Η εφαρμογή προβάλει σημεία και γεωμετρίες από δύο Google Sheets και παρέχει λειτουργίες όπως εντοπισμό θέσης και φιλτράρισμα κοντινών σημείων.
